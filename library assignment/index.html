Image Galleries
an image gallery is a really simple thing, it is an image, in a box, with a description underneath.

Here are a few example to start of with.

As you can see in the above examples an image gallery is exacly what it sounds like, they can each be different sized or you can force them all to be the same size, generally you would want to keep them all the same size both in image actual size and how they are on the website so that you don't get images that look bad due to being shrunk or stretched while others look fine.

Image galleries usually don't display them all at once, usually the description is kept hidden until you click on the image and it gets expanded.
